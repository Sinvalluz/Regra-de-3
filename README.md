# Regra de 3

Este projeto é uma aplicação web simples que resolve problemas de regra de três. É uma ferramenta útil para estudantes, professores e qualquer pessoa que precise realizar cálculos de regra de três de forma rápida e eficiente.

## O que é a Regra de Três?

A regra de três é um método matemático usado para resolver problemas que envolvem quatro valores, onde três são conhecidos e um é desconhecido. Ela se baseia na proporcionalidade entre as grandezas envolvidas.

## Funcionalidades

-   **Cálculo de Regra de Três Simples:** Resolva problemas com duas grandezas diretamente ou inversamente proporcionais.
-   **Interface Intuitiva:** Design limpo e fácil de usar, permitindo que você insira os valores e obtenha o resultado rapidamente.
-   **Responsivo:** A aplicação se adapta a diferentes tamanhos de tela, funcionando bem em desktops, tablets e smartphones.

## Como Usar

1.  Acesse a aplicação através do link abaixo.
2.  Insira os três valores conhecidos nos campos apropriados.
3.  O valor desconhecido será calculado e exibido automaticamente.
